# Libraries

This section provides an overview of the libraries used and developed for this project.

## Custom Libraries

-   **[Standard UART](uart_standard.md):** A cross-platform library for UART communication on ESP32 and AVR microcontrollers.
-   **[Ultrasonic Sensor (DYP-A02YYGDW-V2.0)](ultrasonic_dypa02.md):** A library for the DYP-A02YYGDW-V2.0 ultrasonic sensor.

## Third-Party Libraries

-   **[FreeRTOS](rtos/README.md):** A real-time operating system used for task management.

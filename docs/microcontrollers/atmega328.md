# ATmega328P

The ATmega328P is a popular 8-bit microcontroller used in the Arduino Uno and Nano boards.

## Pin Reference

### Digital Pins

| Pin | Arduino Pin | Function |
| --- | ----------- | -------- |
| PD0 | 0 | RXD |
| PD1 | 1 | TXD |
| PD2 | 2 | INT0 |
| PD3 | 3 | INT1, PWM |
| PD4 | 4 | |
| PD5 | 5 | PWM |
| PD6 | 6 | PWM |
| PD7 | 7 | |
| PB0 | 8 | |
| PB1 | 9 | PWM |
| PB2 | 10 | PWM |
| PB3 | 11 | PWM, MOSI |
| PB4 | 12 | MISO |
| PB5 | 13 | SCK, LED |

### Analog Pins (ADC)

| Pin | Arduino Pin | ADC Channel |
| --- | ----------- | ----------- |
| PC0 | A0 | ADC0 |
| PC1 | A1 | ADC1 |
| PC2 | A2 | ADC2 |
| PC3 | A3 | ADC3 |
| PC4 | A4 | ADC4, SDA |
| PC5 | A5 | ADC5, SCL |

### PWM Pins

Pins 3, 5, 6, 9, 10, and 11 can be used for PWM.
